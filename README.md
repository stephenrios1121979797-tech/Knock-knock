# Knock Knock

Internal door-to-door roofing sales app.

## Purpose
Knock Knock helps roofing sales teams track door activity, avoid re-knocking,
and manage territories across multiple locations.

## Roles
- Manager: oversees reps, locations, and territory status
- Rep: logs door activity in assigned areas

## Door Outcomes
- Knock
- Talk
- Walk (roof only, must follow a talk)
- No Answer

## Rules
- Manual confirmation only
- One outcome per door visit
- Walks cannot be logged without a talk
- No auto-tracking or GPS guessing

## Tech
- Web-based (mobile-friendly)
- Map-based territory view
- Built for internal company use
